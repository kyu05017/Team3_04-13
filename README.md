주제
---

중고차 매매 사이트


***
프로젝트 개요/목적
---
개요 : 중고차 구매 및 판매를 원하는 사람들에게 서비스를 제공한다.

목적 : 판매 목적을 지닌 딜러가 자신의 차량을 등록하고 판매할 수 있게 하고, 구매 목적을 지닌 방문자가 원하는 종류의 차종을 선택하여 딜러와 컨택하여 구매할 수 있게 한다.
